
	Google Chrome Portable by 黯然过客 (64位版)
————————————————

    便携版作者: 黯然过客
    更新时间: 2017/5/11
    发布地址: https://www.portablesoft.org/google-chrome/
    
    基于 Google Chrome 58.0.3029.110 Stable 正式版制作便携版，下载解压后直接运行 ChromePortable.exe 即可，注意现在 Google Chrome 已不支持 Windows XP 了。

    ChromePortable.exe 已完全重写，基本实现了 Chrome 的完全便携化：
    
        1. 支持将任意的命令行参数传递给 Chrome 主程序；
        2. 默认 user-data-dir 目录为 App\Google Chrome\PortableProfile 文件夹；
        3. 可以将 ChromePortable.exe 拖放锁定到任务栏，通过任务栏直接运行 Chrome 便携版；
        4. 可以创建 ChromePortable.exe 的快捷方式，并在快捷方式上设置任意的命令行参数，并运行便携版；
        5. 现在可以将 Chrome 便携版设置为默认浏览器了，Windows 7 系统请直接运行 ChromePortable.bat 脚本完成设置；
        6. 在 Windows 10 中设置默认浏览器的话，执行完 ChromePortable.bat 脚本后，还需要到「设置 > 系统 > 默认应用 > Web 浏览器」中选择 Google Chrome Portable 并确认操作，注意不要使用 Chrome 自带的默认浏览器设置功能。

    Chrome 便携版默认设置了 3 个参数：
    
        1. user-data-dir
        2. no-first-run
        3. no-default-browser-check

    如果你想要自定义 user-data-dir，目前有两种方式：
    
        1. 到 ChromePortable.ini 文件中设置 --user-data-dir 的值；
        2. 创建一个 ChromePortable.exe 的快捷方式，在快捷方式中添加 --user-data-dir=path/to/your/profile  参数，通过快捷方式运行 Chrome 便携版会忽略 ini 文件中的配置。

    添加其他的命令行启动参数：
    
        1. 到 ChromePortable.ini 文件中设置 custom-switches 的值，可以添加任何你需要的参数；
        2. 还是创建一个 ChromePortable.exe 的快捷方式，添加任何你需要的参数。
